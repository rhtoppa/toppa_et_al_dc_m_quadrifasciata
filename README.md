# Homing Ability of *Melipona quadrifasciata* in Fragmented Landscapes

This repository contains the complete analytical pipeline for the study **"Homing ability of *Melipona quadrifasciata* in fragmented landscapes extends to 7.5 km"** published in Discover Conservation.

This research investigates the homing ability of the stingless bee *Melipona quadrifasciata* across fragmented landscapes, demonstrating their remarkable navigation capabilities up to 7.5 km. The study combines survival analysis, generalized linear mixed models (GLMM), and landscape metrics to understand the factors influencing bee return success.
